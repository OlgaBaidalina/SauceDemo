## Чек-лит проверок SauceDemo 

### 1. Страница входа (login Test)

- Вход с валидными данными
- Ошибка при пустом поле Username
- Ошибка при пустом поле Password
- Ошибка при неверном логине и пароле

### 2. Страница с товарами (Products Test)

- Заголовок страницы с товарами
- Обновление счетчика корзины после добавления товара
- Сортировка "Name (A to Z)"
- Сортировка "Name (Z to A)"
- Сортировка "Price (low to high)"
- Сортировка "Price (high to low)"
- Переход на страницу корзины по клику на иконку корзины

### 3. Страница корзины (Cart Test)

- Название добавленного товара совпадает в корзине
- Цена добавленного товара совпадает в корзине 
- Переход на страницу оформления (Checkout)
- Удаление товара из корзины
- Возврат к товарам на страницу с продуктами

### 4. Страница оформления (Checkout Test)

- Ошибки при пустом поле First Name
- Ошибка при пустом поле Last Name
- Ошибка при пустом поле Postal Code
- Переход на Overview после заполнения полей
- Завершение заказа по кнопке "Finish"
